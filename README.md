# Online Retail Sales Dashboard (Power BI)

## Project Overview
This project presents an interactive Power BI dashboard built using an online retail dataset. The dashboard provides insights into sales performance, customer behavior, and geographical distribution of revenue.

## Key Features
- Monthly revenue trend analysis (2011)
- Top 10 countries by revenue and quantity (excluding UK)
- Top 10 customers by revenue
- Global sales distribution using map visualization
- KPI cards for total revenue and total quantity sold

## Tools & Technologies
- Power BI Desktop
- Power Query
- DAX
- Excel Dataset

## Key Learnings
- Data cleaning and transformation
- DAX measure creation using SUMX
- Interactive dashboards and business storytelling
- Data visualization best practices
